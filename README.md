| Language | Special Running Flags |
|----------|-----------------------|
|C++| `-std=c++17 -O3`|

### How to modify flags in code runner
1. VSCode -> Extensions Panel -> Code Runner -> Cog Symbol -> Extension Settings
2. `Code Runner: Executor Map` -> Edit in JSON
